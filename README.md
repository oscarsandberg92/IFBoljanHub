<br/>
#### Setup 
<br>
<ul>
  <lI>
1. Clone the project
  </lI>
  <li>
    2. Start the program, and complete the Umbraco Database installation.
  </li>
  <li>
    3. Go to Settings, and under uSync, choose "Import Everything". 
  </li>
  <li>
    4. Go to Document Types, on the 4'th tab, you will have the option to build the models, press this.
  </li>
  <li>
    5. Close the Browser and stop the program in VS.
  </li>
  <li>
    6. Restart the program, you should now be able to browse the site with full funtionality.
  </li>
</ul>
